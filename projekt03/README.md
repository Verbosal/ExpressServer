# Super-Ankieter
A user-friendliness oriented polling web app that has it all.

## Usage:
Running 
```node index.js```
runs the application @ localhost:8000 (same as 127.0.0.1:8000).

Clicking the first button sends you to a page where you can make your own poll.

Below are all the currently ongoing polls, where you can vote by simply clicking on any option.

Each vote is counted!
